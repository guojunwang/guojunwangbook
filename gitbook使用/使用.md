# Markdown

GitBook默认使用Markdown语法。
下面这些可以作为一个快速参考和展示。更多完整的信息，请参考 John Gruber's original spec 和 Github-flavored Markdown info page。

## 标题
输入
```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```
输出

# H1
## H2
### H3
#### H4
##### H5
###### H6


# 强调

强调：又叫做斜体，使用2个星号或2个下划线包着左右各一个
重点强调:又叫做粗体，使用4个星号或4个下划线包着左右各一个
删除线使用两个波浪线包着，左右各一个

输入
```
*我是斜体*  或  _我是斜体_
**我是粗体** 或 __我是粗体__
~~我是删除线~~
```

对应输出
*我是斜体*  或  _我是斜体_
**我是粗体** 或 __我是粗体__
~~我是删除线~~
